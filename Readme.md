#NeuralTalk - Theano Ver.

작성자: 김택수

NeuralTalk 프로젝트를 theano 버전으로 바꿈.

다른건 오리지날 NeuralTalk과 동일. 

### 추가된 파일들:
* theano_driver.py, 
* theano_driver_minibatch.py 
* theano_test.py
* config.py
* util.py

#### 사용방법
* config.py 적당히 바꿔주기
* Train: theano_driver_minibatch.py에서 train()
* Test: theano_test.py에서 test_main()
